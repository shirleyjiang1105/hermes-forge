# 项目申报书模板

## 触发条件
用户说"写申报书"、"写本子"、"项目申报"时加载。

## Step 1：确认项目类型

| 项目 | 模板 |
|------|------|
| 国自然面上/青年 | 国家自然科学基金模板 |
| 全国教育科学规划 | 教育科学规划模板（偏教育政策） |
| 教育部人文社科 | 人文社科模板（偏理论） |
| 上海市自然科学基金 | 上海市自然基金模板 |
| 上海市教育科学研究 | 上海市教育科研模板 |
| 校级/其他 | 通用学术项目模板 |

## Step 2：生成申报书骨架

### 国自然面上（F0701教育技术）标准结构

```
一、立项依据与研究内容（建议4000-8000字）
  1.1 项目的立项依据
      1.1.1 研究意义（政策需求+实践需求）
      1.1.2 国内外研究现状与分析
      1.1.3 现有研究不足与本项目的切入点
  1.2 项目的研究内容、研究目标及拟解决的关键科学问题
      1.2.1 研究内容（3-4项，逐项展开）
      1.2.2 研究目标
      1.2.3 拟解决的关键科学问题（2-3个，一问一答）
  1.3 拟采取的研究方案及可行性分析
      1.3.1 技术路线（含流程图）
      1.3.2 研究方法（逐项说明）
      1.3.3 可行性分析
  1.4 本项目的特色与创新之处（3点）
  1.5 年度研究计划及预期成果

二、研究基础与工作条件
  2.1 研究基础（前期成果与本项目的关系）
  2.2 工作条件（设备、平台、团队）
  2.3 申请人简历

三、经费预算说明
```

### 教育部/全国教育科学规划 标准结构

```
一、问题的提出
  1.1 政策背景（引用具体文件文号）
  1.2 实践困境（教学一线的真实问题）
  1.3 研究现状与空白

二、研究内容与方法
  2.1 核心概念界定
  2.2 研究内容（3-4项）
  2.3 研究方法与技术路线

三、创新点与预期成果
  3.1 理论创新
  3.2 实践创新
  3.3 预期成果（论文+报告+案例）

四、研究基础与条件
五、研究计划与进度
六、经费预算
```

### 上海市教委/科委 标准结构

```
一、立题依据
二、研究目标与内容
三、创新点
四、研究方法与技术路线
五、预期成果
六、研究基础
七、年度计划
八、经费概算
```

## Step 3：逐板块引导填写

对每个板块：
1. 展示该板块的**写作目标**（审稿人想看什么）
2. 请用户提供**核心信息**（一问一答）
3. 生成**草稿文本**
4. 用户确认后进入下一板块

示例——国自然的"立项依据"板块：

```
=== 板块1：立项依据 ===

写作目标：审稿人3分钟内判断"这个问题值不值得做"
核心要素：政策需求→实践需求→学术空白→你的切入点

请问：
Q1: 本项目回应的国家政策/文件是？（如"教育数字化转型"）
Q2: 教学一线面临的具体问题是什么？
Q3: 文献中尚未解决的核心空白是什么？（一句话）
Q4: 你的切入点是什么？（与前人不同的地方）

→ 生成四段式立项依据草稿
```

## Step 4：自动套用用户的现有信息

从记忆中自动提取以下信息填入骨架：
- 用户学术身份、单位、研究方向
- 现有SCI论文列表（5篇）
- 在研项目（市教委AI重点、暮盒横向）
- 已有设备（16导联便携EEG）
- 教育部协同创新课题经历
- 五年总纲中的研究方向

标注"[自动填充]"和"请用户确认"。

## Step 5：输出

生成两份文件：
1. `申报书骨架-{项目类型}-{日期}.md`：空模板+写作提示
2. `申报书草稿-{项目类型}-{日期}.md`：填充用户信息后的初稿

## 注意事项
- 政策文件引用必须注明文号，不编造
- 研究内容"3-4项"是铁律——不能多（散）不能少（单薄）
- 创新点不超过3条，每条必须有"前人没做到→我们做到了"的对比
- 经费预算按学校财务标准，不随意写
